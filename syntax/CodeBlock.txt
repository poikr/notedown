코드 블록

이 문서는 코드 블록에 대한 구문을 설명합니다.

코드 블록은 일반적으로 프로그래밍 코드나 명령어를 포함하는 텍스트 블록을 나타냅니다. 코드 블록은 다음과 같은 방식으로 작성할 수 있습니다:

```python
def hello_world():
    print("Hello, World!")
```

위 예제에서, 코드 블록은 세 개의 백틱(```)으로 시작하고 끝납니다. 첫 번째 백틱 뒤에 언어 이름(예: python)을 지정하여 언어를 명시할 수 있습니다. 이렇게 하면 코드 하이라이팅이 적용될 수 있습니다.

코드 블록 내에서는 들여쓰기, 공백, 특수 문자 등이 그대로 유지됩니다. 따라서 코드의 형식이 중요할 때 유용합니다.

2. 특수 언어

iframe을 언어로 지정하면 HTML iframe 코드 블록을 생성할 수 있습니다.
iframe 코드블록 내부의 내용은 HTML로 해석됩니다.

기본 예시:
```iframe
<div style="background: red;">안녕하세요</div>
<button onclick="alert('Hello!')">클릭하세요</button>
```

iframe 속성 지정:
iframe 코드 블록에는 다음과 같은 속성을 지정할 수 있습니다:
- w# : width (너비) 지정 (예: w#300px, w#100%)
- h# : height (높이) 지정 (예: h#400px, h#50vh)
- r# : resize (크기 조절) 설정
  - r#w : width만 조절 가능
  - r#h : height만 조절 가능
  - r#b : 양방향(both) 조절 가능
  - r#n : 조절 불가(none)

예시:
```iframe,w#500px,h#300px,r#b
<h1>크기 조절 가능한 iframe</h1>
<p>이 iframe은 500px 너비, 300px 높이로 시작하며, 양방향으로 크기 조절이 가능합니다.</p>
```

```iframe,w#100%,r#n
<div>크기 조절이 불가능한 전체 너비 iframe</div>
```

```iframe,h#200px,r#h
<div>세로 크기만 조절 가능한 iframe</div>
```
