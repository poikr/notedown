텍스트 데코레이션

이 문서는 텍스트 데코레이션에 대한 구문을 설명합니다.

1. Bold

굵은 텍스트를 만들기 위해서는 텍스트 양쪽에 두 개의 별표(**)를 사용합니다.
예시:
**이 텍스트는 굵게 표시됩니다.**

2. Italic

기울임꼴 텍스트를 만들기 위해서는 텍스트 양쪽에 하나의 별표(*)를 사용합니다.
예시:
*이 텍스트는 기울임꼴로 표시됩니다.*

3. Underline

밑줄 텍스트를 만들기 위해서는 텍스트 양쪽에 밑줄 기호(__)를 사용합니다.
예시:
__이 텍스트는 밑줄이 그어집니다.__

4. Strikethrough

취소선 텍스트를 만들기 위해서는 텍스트 양쪽에 물결표(~~)를 사용합니다.
예시:
~~이 텍스트는 취소선이 그어집니다.~~

5. Combined Decorations

여러 데코레이션을 결합할 수도 있습니다. 예를 들어, 굵은 기울임꼴 텍스트를 만들려면 다음과 같이 작성합니다.
***이 텍스트는 굵고 기울임꼴로 표시됩니다.***
**__이 텍스트는 굵고 밑줄이 그어집니다.__ *이 텍스트는 굵고 이탈렉체로 표기됩니다.***

6. Colored Text

텍스트 색상을 변경하려면 다음과 같은 구문을 사용합니다.
Tag가 없을경우 전경으로 설정됩니다.
|color,text|
예시:
|red,이 텍스트는 빨간색으로 표시됩니다.|

f# 태그가 있을경우 전경으로 설정됩니다.
|f#color,text|
예시:
|f#00FF00,이 텍스트는 녹색으로 표시됩니다.|
|f#red,이 텍스트는 빨간색으로 표시됩니다.|

b# 태그가 있을경우 배경으로 설정됩니다.
|b#color,text|
예시:
|b#0000FF,이 텍스트는 파란색 배경으로 표시됩니다.|
|b#yellow,이 텍스트는 노란색 배경으로 표시됩니다.|

,를 사용하여 전경색과 배경색을 동시에 설정할 수도 있습니다.
|f#color1,b#color2,text|
|b#color2,f#color1,text|
예시:
|f#yellow,b#green,이 텍스트는 노란색 전경과 녹색 배경으로 표시됩니다.|
|b#black,f#white,이 텍스트는 흰색 전경과 검은색 배경으로 표시됩니다.|

\를 사용하여 색상 데코레이션을 이스케이프할 수 있습니다.
예시:

a.
\|red,이 텍스트는 색상이 적용되지 않습니다.\| 
=> 표기 결과 : |red,이 텍스트는 색상이 적용되지 않습니다.|

b.
|\f#00FF00,이 텍스트는 색상이 적용되지 않습니다.|
=> 표기 결과 : |f#00FF00,이 텍스트는 색상이 적용되지 않습니다.|

c.
|\b#0000FF,이 텍스트는 색상이 적용되지 않습니다.|
=> 표기 결과 : |b#0000FF,이 텍스트는 색상이 적용되지 않습니다.|

7. Code Block

`를 활용하여 인라인 코드 블록을 만들 수 있습니다.
코드 블록 내에서는 다른 데코레이션이 적용되지 않습니다.
그러나 코드 블록 밖에서 적용한 데코레이션은 코드 블록 내부에 영향을 미칩니다.
예시:
`이 텍스트는 코드 블록입니다.`
**이 텍스트는 굵게 표시되고 `이 코드 블록은 굵게 표시됩니다.`**

8. LaTeX Support

$를 활용하여 LaTeX 수식을 작성할 수 있습니다.
수식 내에서는 다른 데코레이션이 적용되지 않습니다.
예시:

a. 
$E=mc^2$

b. 
이러한 방식으로 수식을 작성할 수 있습니다: $a^2 + b^2 = c^2$.
=> 표기 결과 : 이러한 방식으로 수식을 작성할 수 있습니다: a^2 + b^2 = c^2.

***굵고 기울임꼴 텍스트***

|b#black,f#white,흰색 텍스트와 검은색 배경|
\|red,색상 적용 안됨\|

\|\b#0000FF,색상 적용 안됨\|
